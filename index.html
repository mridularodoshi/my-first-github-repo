What is Git?

In short, Git is a version control system that allows multiple developers to collaborate on projects efficiently by managing changes to source code over time. It's fast, supports branching and merging, ensures data integrity, and is widely used in software development due to its effectiveness in managing and tracking code changes.

Git vs Github

In essence, Git is the version control system itself, whereas GitHub is a platform built around Git, offering additional features to enhance collaboration and project management. Developers use Git to manage their source code locally, and they can use GitHub (or similar platforms like GitLab, Bitbucket) to host their Git repositories remotely and facilitate team collaboration.

How to push incremental codes to Github:

  • git add .
  • git commit -m"add a meaningful/relevant commit message"
  • git push
